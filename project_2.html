The ASCIII character set is a type of code that uses 7 bits that represent character data. Explain the storage formats for different types of digital data (character, numeric, audio and video).

Digital data is stored in digital files that exist on a storage medium such as CD, DVD, or flash drive. Each file contains an extension that indicates the file format, the type of data that exists within the file. Numeric data is stored in the binary number system, which allows the computer to read and calculate the numbers used in arithmetic operations. Character data, also referred to as text, is an encoding format that uses codes of 7 bits to represent character data. The data are represented as 1’s and 0’ s, carried out as pulses of a current through a circuit board, and stored on CD’s or DVD’s and light and dark spots, or on disk drives as magnetized particles. (Ch. 1 – pg. 18-20) Digital audio is stored in different file formats with distinct advantages, and disadvantages. In order to store an audio file, it is sampled by an ADC to convert the sound into binary 1’s and 0’s, then it is stored as bits, compressed, and stored into audio formats such as AAC, MP3, Vorbis, WAV, and WMA. (Ch. 1 – pgs. 28-30) Video files are stored in container formats that contain the compressed video stream and the audio stream. Popular video container formats include AVI, MOV, MPEG, WebM, ASF, VOB, and Ogg Theora. (Ch. 1 – pg. 62)

  1. Comparing Hardware

    • There are multiple types of hardware that a digital device contains in order to function properly, each with their own task to ensure the device is running properly. RAM, Random Access Memory, is a type of computer hardware used for storage processing of information. RAM has a much faster processing speed than a hard disk and is a volatile device that stores data temporarily until the device system is shut down, when it wipes out the information that was stored in it.
    • A hard disk is another computer hardware and stores data in a different way. The hard disk is either preinstalled into the CPU or can be used as an external device. Unlike RAM, the hard disk is non-volatile and stores data permanently, so it does not wipe out the information when the system is shut down. It has a large capacity to story trillions of bytes of data that can be accessed easily. The hard disk has an electromagnetic surface in order to store data and has various rpms that determine the processing speed of the hard disk.

  3. VPN

    • VPN, Virtual Private Network, is the opportunity to obtain a protected network connection when using public networks. A VPN can encode internet traffic and conceal online identity making it more difficult for a third party to track or steal your online data. A VPN hides the IP address of a device by letting the network redirect it into a remote server run by the VPN host, making it so that your internet service provider or other third parties cannot see your online history or the data you send and receive online. The data is run through a filter that turns it into “gibberish” to someone trying to obtain your data. VPN connection can secure encryption in your device, disguise whereabouts, give access to regional content not always available, and secure data transfer. However, there are possible downsides when using a VPN. The level of privacy, for example, is dependent on the VPN service you obtain and how trustworthy it is. One of the main aspects of VPN service is privacy, therefore, the best option would be to incorporate a no-logs policy, or a VPN is threatening and can be unsafe. In addition, using a VPN can slow down connection speed since it is directly connected to the internet. With a good quality VPN, the speed loss should be minimal, but it’s primary focus is on protection of privacy which can lead to slow download speeds.
